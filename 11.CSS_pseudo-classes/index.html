Một số lớp giả hay dùng

  1. :root : Tham chiếu đến 1 phần từ trong chính file html.
  2. :hover : Kích hoạt trạng thái CSS đặc biệt của 1 phần tử khi được di chuột vào.
  3. :active : Kích hoạt trạng thái CSS đặc biệt của 1 phần tử khi được click và giữ chuột, giả chuột sẽ mất tác dụng
  4. :first-child : CSS đến phần tử con đầu tiên trong 1 tập hợp các phần tử
  5. :last-child : CSS đến phần tử con cuối cùng trong 1 tập hợp các phần tử

Ta đặt 1 lớp giả

:root { --text-color: green }

Từ giờ giá trị của biến --text-color sẽ có màu green. Khi cần gọi lại giá trị biến đó. Ta ghi thuộc tính và gọi ra biến đó như sau:

color: var(--text-color)

Ví dụ

:root {
--text-color: green;
}

#paragraph1{
color: var(--text-color)
}

#paragraph2{
color: var(--text-color)
}

#paragraph3{
color: var(--text-color)
}

Sau này khi ta thay đổi giá trị của biến --text-color thì tất cả những nơi có gọi đến biến --text-color đều được đổi màu

Ví dụ

#box-1:hover {
background-color: red;
color: rgb(0, 4, 255);
font-size: 30px;
}


Di chuột vào đây

Ví dụ

#box-2:active {
background-color: red;
color: rgb(0, 4, 255);
font-size: 30px;
}

Click chuột vào đây

Ví dụ first-child

li.exfirstchild-list:first-child
{
color:red
}

  1. Item đầu tiên
  2. Item thứ 2
  3. Item thứ 3
  4. item thứ 4
  5. Item cuối cùng

Ví dụ: last-child

li.lastchild-list:last-child
{
color:blue
}

  1. Item đầu tiên
  2. Item thứ 2
  3. Item thứ 3
  4. item thứ 4
  5. Item cuối cùng
