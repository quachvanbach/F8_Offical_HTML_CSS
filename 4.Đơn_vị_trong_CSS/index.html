Các đơn vị thường sử dụng trong CSS

    I. Đơn vị tuyệt đối

  1. px: với màn hình có độ phân giải thấp thì 1 px tưởng ứng với 1 điểm ảnh thực, còn với màn hình độ phân giải cao thì 1 px sẽ chiếm nhiều điểm ảnh h

    2. II. Đơn vị tương đối

  2. %: tương ứng với % của thẻ chứa đối tượng được css
  3. rem: đối tượng được CSS sẽ phụ thuộc vào thẻ html
  4. em: phụ thuộc vào thẻ gần nhất chứa đối tượng có thuộc tính css
  5. vw/vh (viewport width/viewport height): 1vw/vh sẽ tương ứng với 1% chiều ngang/dọc của màn hình hiển thị
