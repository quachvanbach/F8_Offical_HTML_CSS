}

Giống và khác nhau giữa Declaration function và Expression function

Giống:

- Đều có tính chất private

- Không chạy khi định nghĩa

Khác:

- Declaration function: Có thể gọi trước khi định nghĩa

- Expression function: Có thể đặt tên hoặc không. Đặt tên chỉ với mục đích làm rõ ý nghĩa sử dụng.
