I. Arithmetic operators (Toán tử số học)

Toán tử số học
Tên toán tử Toán tử Cách trình bày
Cộng + a + b
Trừ - a - b
Nhân * a * b
Luỹ thừa ** a ** b
Chia / a / b
Chia lấy số dư % a % b
Tăng 1 giá trị số ++ a++
Giảm 1 giá trị số -- a--

II. Assignment Operator (Toán tử gán)

Toán tử Ví dụ Tương đương Ví dụ minh hoạ
= x = y x = y a = 2
+= x += y x = x + y a += 2
-= x -= y x = x - y a -= 2
*= x *= y x = x * y a *= 2
/= x /= y x = x / y a /= 2
**= x **= y x = x ** y a **= 2

III. Comparison Operator (Toán tử So sánh)

Toán tử Ý nghĩa
== So sánh Bằng
=== So sánh Bằng (tuyệt đối). So sánh cả về giá trị và kiểu dữ liệu.
!= So sánh Không bằng
!== So sánh Không bằng (Tuyệt đối). So sánh cả về giá trị và kiểu dữ liệu.
> So sánh Lớn hơn
<< So sánh Nhỏ hơn
>= So sánh Lớn hơn hoặc Bằng
<= So sánh Nhỏ hơn hoặc Bằng

IV. Logical operators (Toán tử Logic)

Toán tử ý nghĩa
&& And (và)
|| Or (hoặc)
! Not (phủ định, ngược lại)

V. Notice when using logical operators with if-else. (Lưu ý khi sử dụng toán tử logic với if)

- Với toán tử && thì sẽ xét lần lượt từ phải qua trái và nếu tìm ra giá trị là 1 trong 6 giá trị false và in nó ra.

- Nếu không có giá trị false thì in ra giá trị tận cùng bên phải của chuỗi.



- Với toán tử || thì sẽ xét lần lượt từ trái qua phải và bỏ qua giá trị là 1 trong 6 giá trị false. Nếu trong quá trình xét, gặp được giá trị khác giá trị false thì in nó ra

- Nếu toàn bộ giá trị là false thì in giá trị cuối cùng bên phải
