I. Primitive (Dữ liệu nguyên thuỷ)

Kiểu dữ liệu Ý nghĩa Ví dụ
Number Kiểu số

var a = 1;

var c = 3;
String Kiểu chuỗi

var lastName = 'Bach';

var firstNam = 'Quach';

var fullName = 'Quach Van Bach';
Boolean Thể hiện giá trị True hoặc False

var isSuccess = true;

var isFalse = false;

Null Thể hiện không có gì, không có gì xảy ra var isNull = null

Complex Data (Kiểu dữ liệu phức tạp)

Kiểu dữ liệu Ý nghĩa Ví dụ Gọi ra giá trị
Function Function chỉ được thực thi khi ta gọi đến nó. var myFunction = function() {
alert('Hello world')
} 		myFunction()
Object Object Truyền vào được nhiều cặp giá trị có cú pháp: key: 'value' . Các cặp giá trị cách nhau bằng dấu phẩy, hoặc có thể truyền vào 1 function var myObject = {
key: 'value',
name: 'Bach',
age: '27',
myFunction: function() {
} 		console.log(myObject)
Array Truyền vào 1 danh sách các giá trị. Key sẽ được đánh tự động theo thứ tự tăng dần. var myArray = [
'Javascript'
'PHP'
'Ruby'
]		 console.log(myArray)
