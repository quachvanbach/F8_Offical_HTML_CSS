Relative

.h1-relative {
position: relative;
top: 200px;
}

Với thuộc tính "position: relative". Phần tử được CSS sẽ lấy chính vị trí của nó đang đứng làm gốc toạ độ. Và t có thể tự do css vị trí của nó. Có thể cho nó đè lên phần tử khác.

Absolute

Với thuộc tính "position: absolute". Phần tử được CSS sẽ bị phụ thuộc vào thẻ cha gần nhất có chứa thuộc tính position làm gốc toạ độ.

.box-dad {
width:100%;
height:450px;
border: solid 1px green ;
background-color: rgb(154, 129, 241);
position: relative;
}

.box-child {
width: 200px;
height: 200px;
background-color: blue;
position: absolute;
top: 0px;
right: 0px;
}

Fixed

Với thuộc tính "position: fixed. Phần tử được CSS sẽ lấy chính cửa sổ trình duyệt làm gốc toạ độ. Và sẽ được ghim tại vị trí đó khi cuộn cửa sổ trình duyệt"

.box-fixed{
height:fit-content;{
background: rgb(97, 106, 228);{
position: fixed;{
bottom:0;{
left:0;{
right: 0;{
}

Sticky

Ngoài ra còn có thuộc tính "position: sticky". Nhưng vì nó còn hạn chế trình duyệt nên ít được sử dụng.
