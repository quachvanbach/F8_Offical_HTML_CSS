Một số hàm built-in

  • 1. alert : Bật lên cửa sổ hộp thoại
  • 2. console : In ra giá trị trong cửa số console của trình duyệt
  • 3. confirm : Bật cửa sổ hộp thoại có kèm nút xác nhận
  • 4. promnt : Bật cửa sổ hộp thoại có ô input và nút xác nhận giá trị nhập
  • 5. setTimeout : Chạy 1 đoạn code sau khoảng thời gian được cài đặt
  • 6. setInerval : Chạy 1 đoạn code liên tục sau khoảng thời gian được cài đặt.
