Các cách tạo chuỗi

Cách 1

var fullName = 'Quach Bach'

Cách 2

var fullName = new String('Quach Bach')

Không nên sử dụng cách thứ 2 vì cách viết dài dòng, kiểu dữ liệu được tạo ra là kiểu Object.

Một số trường hợp sử dụng Backslash (\)

var fullName = 'Đây là dấu ''

var fullName = "Đây là dấu \""

var fullName = 'Đây là dấu \\'

Sử dụng dấu Backslash \ trước dầu nháy đơn (') hoặc dấu nháy kép (") hoặc dấu backslash (\) để định nghĩa dấu đó trong chuỗi.

Xem độ dài của chuỗi

var fullName = 'Quach Bach'

console.log(fullName.length)

Chú ý độ dài khi viết code

Nên sử dụng tối đa 80 ký tự trên 1 dòng.

Template string

var firstName = 'Quach'; var lastName = 'Bach';

Thay vì sử dụng:

console.log('Tôi là: ' + firstName + ' ' + lastName)

Ta có thể viết:

console.log(`Tôi là: ${firstName} ${lastName}`)
