::before : Css cho phần sau của 1 thẻ.

.box::before {
content: "";
display: block;
width:100px;
height: 100px;
background-color: rgb(255, 0, 0);
}

::after : Css cho phần sau của 1 thẻ.

.box::after {
content:"";
display: block;
width: 200px;
height:200px;
background-color: blue;
}

::first-letter : Css cho chữ cái đầu tiên của thẻ

Cú pháp CSS:
p.letter::first-letter{
color: blue;
font-size: 50px;
}

::first-line : Css cho dòng đầu tiên của thẻ.

Cú pháp CSS:
p.line::first-line{
color: blue;
font-size: 50px;
}

::selection : Css phần được lựa chọn ( bôi đen )

Lựa chọn ( bôi đen ) phần văn bản bên dưới đây:

Cú pháp CSS:
p.selection::selection{
color: blue;
font-size: 50px;
}
